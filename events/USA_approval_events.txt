# ==============================================================
#  USA APPROVAL - OVERSHOOT EVENTS
#
#  Fourteen narration events, fired by USA_apr_fire_high_event
#  and USA_apr_fire_low_event when approval lands more than
#  usa_apr_event_at points outside the band.
#
#  They carry NO effects on purpose. The daily bleed has already
#  started moving the number before the player opens the popup;
#  these only give the movement a reason.
#
#      usa_approval.1  - .7   approval was too HIGH, knocked back
#      usa_approval.11 - .17  approval was too LOW, recovering
#
#  Pictures are vanilla placeholders. A missing sprite shows an
#  empty frame rather than crashing, so swap freely.
# ==============================================================
